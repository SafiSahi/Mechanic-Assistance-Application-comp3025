# MechConnect: Mechanic Assistance Application

## Overview
MechConnect is a mobile app that helps users find and book nearby mechanics for real-time, on-demand vehicle assistance. It connects users with verified mechanics through location-based services.

## Objectives
- Help users locate the nearest verified mechanic quickly.
- Allow users to request services and get cost estimates.
- Enable mechanics to manage requests, jobs, and profiles.
- Build trust via ratings, reviews, and service history.

## Key Features
- User and mechanic login (Email/Google)
- Booking and job tracking
- Mechanic profiles, pricing, and availability
- Service request and confirmation
- User profile and settings
- Review and rating system

## Technologies
- Kotlin
- Android Studio
- Firebase (Auth + Firestore)
- Google Maps API
- Jetpack Compose (UI)
